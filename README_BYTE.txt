El proyecto esta creado con el uso del framework Angular, por lo tanto para poder correrlo se debe instalar nodeJS de su pagina oficial,
Dejo el enlace: https://nodejs.org/es/

Lo instalamos, lo descargamos y procedemos a instalar angular: 
Vamos a la pagina oficial de angular: https://angular.io
Seguimos los pasos para poder instalarlo y luego cuando tengamos todo junto, en la carpeta raiz ejecutamos desde la consola el comando:

ng serve
o
ng serve -o

el proyecto se compilara y sera mostrado en el navegador en el puerto 4100, en el caso de que no se ejecute automaticamente en
el navegador, pegue la siguiente direccion: http://localhost:4200

Luego de esto puede proceder a probar la aplicacion web SPA.
